echo
Lession 1

Like console.log() in JavaScript, printf() in C and cout<< in C++echo is used in PHP to output something to the page

Sample echo code:
echo "Hi";
echo "Hello";

Output page:
HiHello

Lession 2

PHP code is executed on the server and only the output is sent to the client browser.You can echo a <br> tag to make a line break in your output page like below:

Sample echo code:
echo "Hi";
echo "<br>";
echo "Hello";

Output page:
Hi
Hello

You can also do the same in a single echo like below:

Sample echo code:
echo "Hi<br>Hello";

Output page:
Hi
Hello

Lession 3

You can perform any operations on the echo statement and the output will be printed to the page. Note: The operations written inside <?php and ?> will be executed on the sever and hence it will not be visible on the final output code the browser receives

Sample echo code:
echo 1+2+3+4+4+5+6+7+8+9; // Mathematical operations
echo "<br>";
echo "Hi" . " Hello";
// Concatenation is done with . instead of + in other languages

Output page:
49
Hi Hello

Note: Unlike prompt() in JavaScript, scanf() in C and cin>> in C++ you don't have any function in PHP that could accept user input because PHP code is executed on the server when a page is loading. User inputs are received in PHP server side code through the <form></form> tags and their <input> tags in html code. When an HTML form is filled and submited by the user we usually simply redirect to another file mentioned in the "action" attribe of form tag. Here while redirecting to the action page, the form data is also sent to the server. You'll see how to read those data from PHP code later on.

Variables
Lession 1: Types of variables

Variables in PHP is much simpler than variables in any other programming languages.
In C programming language you write variables like below:

int minAge = 20;
float PI = 3.14159;
string name = "Jishnu Raj";
......

In JavaScript you write variables like below:

var minAge = 20;
var PI = 3.14159;
var name = "Jishnu Raj";

Where as in PHP defining variables is much simpler. Take a look at this code:

Sample echo code:
$minAge = 18; // Integer
$PI = 3.14157; // Float
$user_name = "Jishnu Raj"; // String
$isLoggedIn = true; // Boolean
echo $user_name;

Output page:
Jishnu Raj
Strings
Lession 1: String functions

String is just one of the datatypes in PHP like integer, float, boolean etc. but there are a lot of functions available for performing numerous operations on strings. Some of them are:

  • String concatenation
  • String length
  • Trim
  • Sub string
String concatenation

Multiple strings are concatenated (joined) together using the . operator. Sample code:

$first_name = "Jishnu";
$last_name = "Raj";
$full_name = $first_name . " " . $last_name;
echo $full_name;

String length

To find the number of characters in a string you can use strlen() function. Sample code:

$name = "PHP";
$name_length = strlen($name);
echo "Number of characters in name variable is: ".$name_length;
echo "<br>";
echo "Number of characters in the word 'elephant' is " . strlen("elephant");

Trim:

trim() function is used to remove whitespace or other characters from both sides of a string. Sample code:

$name = " Jishnu Raj ";
echo $name;
echo "<br>";
echo trim($name);

Output page:
Jishnu Raj
Jishnu Raj

Sub string:

substr() function returns a part of a string.
syntax: substr(string,start,length)

$name = "Jishnu Raj";
echo substr($name, 7);
echo "<br>";
echo substr($name, 0, 7);
echo "<br>";
echo substr($name, 7, 2);

Output page:
Raj
Jishnu
Ra
